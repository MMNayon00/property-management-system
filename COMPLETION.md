## 🏗️ প্রযুক্তি স্ট্যাক

### ফ্রন্টএন্ড
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + ShadCN UI
- **State**: React Query + Zustand
- **Form**: React Hook Form + Zod

### ব্যাকএন্ড
- **Runtime**: Node.js 20+
- **Framework**: Next.js API Routes
- **Database**: PostgreSQL 14+
- **ORM**: Prisma
- **Auth**: NextAuth.js
- **Validation**: Zod

### ডেভপস
- **Containerization**: Docker + Docker Compose
- **Web Server**: Nginx
- **Security**: SSL/TLS, Security Headers, Rate Limiting

## 🚀 দ্রুত শুরু

### 1. ডেভেলপমেন্ট সেটআপ

```bash
# প্রয়োজনীয়তা: Node.js 18+, PostgreSQL 14+

# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# পরিবেশ ভেরিয়েবল সেট করুন
cp .env.example .env.local

# ডাটাবেস স্কিমা প্রস্তুত করুন
npx prisma migrate dev --name init

# ডেভেলপমেন্ট সার্ভার চালু করুন
npm run dev
```

### 2. অ্যাক্সেস

- **অ্যাপ্লিকেশন**: http://localhost:3000
- **প্রথম অ্যাডমিন তৈরি করুন**: আমাদের `SETUP.md` দেখুন

### 3. ডকার ডেপ্লয়মেন্ট

```bash
# সমস্ত সেবা চালু করুন
docker-compose up --build

# অ্যাপ্লিকেশন চলছে: http://localhost
```

## 📋 এপিআই এন্ডপয়েন্ট

### অথেন্টিকেশন
- `POST /api/auth/signup` - নতুন অ্যাকাউন্ট তৈরি
- `POST /api/auth/[...nextauth]` - লগইন/সেশন

### অ্যাডমিন
- `GET /api/admin/pending-users` - অপেক্ষমাণ ব্যবহারকারী
- `POST /api/admin/approve-user` - অনুমোদন/প্রত্যাখ্যান
- `GET /api/admin/stats` - ড্যাশবোর্ড পরিসংখ্যান

### সম্পত্তি ব্যবস্থাপনা
- `GET /api/buildings` - বাড়ি তালিকা
- `POST /api/buildings` - নতুন বাড়ি
- `GET /api/flats?buildingId=X` - ফ্ল্যাট তালিকা
- `POST /api/flats` - নতুন ফ্ল্যাট
- `GET /api/tenants?flatId=X` - ভাড়াটিয়া তালিকা
- `POST /api/tenants` - নতুন ভাড়াটিয়া

### আর্থিক
- `GET /api/rent?flatId=X&month=Y` - ভাড়া রেকর্ড
- `POST /api/rent` - নতুন ভাড়া রেকর্ড
- `GET /api/payments` - পেমেন্ট তালিকা
- `POST /api/payments` - পেমেন্ট রেকর্ড করুন

### রিপোর্টিং
- `GET /api/reports?type=monthly` - মাসিক রিপোর্ট
- `GET /api/reports?type=tenant` - টেন্যান্ট ইতিহাস

বিস্তারিত API ডকুমেন্টেশন দেখুন: [API.md](./API.md)
